Минимальное руковдство пользователя:
1. Запускается программа путем запуска main.py. Требуемые библиотеки: pygame, pygame-menu.
2. Навигация по экрану осуществляется с помощью клавиш W A S D.
3. Изменение скорости расчетов осуществляется при помощи клавиш: Стрелочка влево, Стрелочка вправо, либо при нажатии на экранные кнопки.
4. Планету можно отследить. Для этого нужно нажать на клавину Y, переключаться по списку спутников можно на клавиши Q (вперед) и R (назад)
5. Можно менять "центр" системы на конкретные планеты. 1 - Солнце, 2 - Меркурий, 3 - Венера, 4 - Земля, 5 - Марс, 6 - Юпитер, 7 - Сатурн, 8 - Уран, 9 - Нептун.
